### Cách dùng Reset this PC trên Windows 11

1. Mở **Settings**
2. Vào **System** → **Recovery**
3. Tại mục **Recovery options**, tìm **Reset this PC**
4. Nhấn **Reset PC**

Sau đó sẽ có 2 lựa chọn:

* **Keep my files** → Giữ lại file cá nhân (Documents, Desktop, Pictures…), nhưng **xóa toàn bộ ứng dụng và cài lại Windows**
* **Remove everything** → **Xóa sạch toàn bộ dữ liệu và app**, cài lại Windows như máy mới

### Sau đó Windows sẽ hỏi thêm:

* **Cloud download** → Tải bản cài mới từ internet (~4–5GB)
* **Local reinstall** → Cài lại từ file Windows sẵn trong máy

Máy sẽ tự **restart và cài lại Windows tự động**.

✅ **Kết luận:**
Hướng dẫn bạn viết **vẫn đúng với Windows 11**, chỉ cần sửa đường dẫn:

* **Windows 10:** Settings → Update & Security → Recovery
* **Windows 11:** Settings → System → Recovery

💡 Nếu bạn muốn, mình có thể chỉ thêm:

* Khi nào nên **Reset Windows**
* Khi nào nên **cài lại Windows bằng USB** (sạch hơn và ít lỗi hơn).
